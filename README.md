# 最简单的py to exe方法
### 运行方式：
1、使用pip安装py2exe库 
```cmd
pip install py2exe
```
2、将需要转换的.py文件放在build.py放置的文件夹。  
3、打开build.py。  
4、输入文件名（去掉.py后的）。    
5、窗口关闭后，打开dist文件夹，里边就是exe文件包。
